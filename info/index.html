PCでのシリアル・モデル番号表示方法

1. 上記の文字を選択 / 2. [右クリック]して、[コピー]をクリック
3. [Windowsキー]を押しながら、[R]のキーを押す ※[ファイル名を指定して実行] ダイアログ ボックスが開く
4. [ファイル名を指定して実行]内の入力欄で[右クリック]して、[貼り付け]をクリック ※文字を貼り付ける
5. [OK]をクリック ※コマンドプロンプトが表示される。
6. 表示内容
IdentifyingNumber Name
